# 赵龙杰

## 个人概述

10 年产品研发经验，在企业数字化架构和团队管理方面展现了卓越的领导能力和创新精神。成功推动了多个系统（自研低代码平台搭建了行业全流程系统，数据质量管理平台，Salesforce 迁移，Netsuite 迁移，AWS 云原生应用构建）的开发，显著提高了开发效率并降低了成本。

敏捷联盟认证敏捷教练，具备出色的跨部门协作能力，能够有效整合资源，在担任敏捷教练期间，引领团队完成技术转型和流程优化，大幅提升了项目交付的速度和质量，实现复杂项目的顺利推进。
